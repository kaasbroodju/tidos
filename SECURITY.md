# Security Policy

## Supported Versions

Tidos is currently in a pre-release phase. Only the latest version receives security updates.

| Version | Supported |
| ------- | --------- |
| latest  | ✅        |
| older   | ❌        |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub Issues.**

Instead, use [GitHub's Private Vulnerability Reporting](https://github.com/kaasbroodju/tidos/security/advisories/new) to submit your report confidentially.

Your report should include:
- A description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested mitigations (optional)

## What to Expect

Once your report is received, I will send an acknowledgement so you know it has been read.

Fixes will be released as a patch version and disclosed via a [GitHub Security Advisory](https://github.com/kaasbroodju/tidos/security/advisories).
